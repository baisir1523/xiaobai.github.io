# 题目描述

你是一个专业的小偷，计划偷窃沿街的房屋，每间房内都藏有一定的现金。这个地方所有的房屋都 围成一圈 ，这意味着第一个房屋和最后一个房屋是紧挨着的。同时，相邻的房屋装有相互连通的防盗系统，如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警 。

给定一个代表每个房屋存放金额的非负整数数组，计算你 在不触动警报装置的情况下 ，今晚能够偷窃到的最高金额。

# 解题思路

​ 首先他和打家劫舍 I 一样，只不过变成了环形的。我们都知道，一条线变成环可以让其首尾相连。而我们可以以此为本题的突破口。

​ 假象一下，如果我们在选择偷那座房屋的时候，我们直接不考虑开始的屋子或者结束的屋子，也就是说假设如果我们在偷的过程中，故意不偷开始的屋子或者结束的屋子，这样就可以防止同时偷了开始和结束的屋子导致连成环。因为你在偷的过程中故意避开了开头或结尾的房屋。

​ 我们将其分为两种情况（有 n 个房子）：

​ 不偷开始的房子：我们直接偷第二到最后一间房子（下标范围 1~n），然后通过将其简化成了打家劫舍 I 中的情况。

​ 不偷结尾的房子：我们直接偷第一个到第 n-1 个房子（下标范围 1~n-1），然后也将其成功简化为了打家劫舍 I 中的情况。

​ 然后我们将上面的两情况取最大值就是我们想要的结果了。

# dp 数组的定义，含义及其初始化

​ dp [i] 表示前 i 个房间可以偷得的最大价值。dp 数组的起始值为要偷的房子中的第一个房子的价值，dp 数组的起始值 + 1 为要偷的房子中的第一个房子和第二个房子的价值的最大值。

# 状态转移方程

​ dp [i] 是根据 dp [i-1] 和 dp [i-2] 的状态决定

​ 如果偷当前房间：当前房间的总价值为：当前房间的价值加上 dp [i-2] 的价值，然后和 dp [i-1] 的价值进行比较，取最大值。

​ 如果不偷当前房间：则当前房间的总价值为：dp [i-1]

​ 于是方程为：

​ dp[i] = max(dp[i-1] , dp[i-1] + nums[i]);

# 遍历顺序

​ 因为 dp [i] 是根据 dp [i-1] 和 dp [i-2] 的状态决定，所以我们依然从 2 开始向后遍历。

# 完整题解代码

​ 我们这里直接通过调用函数方法来解决问题，函数是打家劫舍 I 中的函数，只不过多了两个形参，首先 nums 为系统的 nums。start 和 endl 是我们传递的下标范围；如果我们不偷第一个房间，则传递 1 和 n；如果不偷最后一个房间则传递 1 和 n-1。

​ 同时我们也考虑了一些特殊情况（没有房间给你偷，或者只有一间房等情况）
