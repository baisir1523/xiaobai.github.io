# 什么是二叉树？

​ 二叉树是一种常见的树状数据结构，由节点（Node）和连接它们的边组成。每个节点最多能够有两个子节点，分别称为左孩子（Left Child）和右孩子（Right Child）。

​ 二叉树的形状就像一个倒着的树，树根在最顶端。如图所示

​ 以下是二叉树的一些名词解释：

​ 根节点：这是二叉树的起始节点，它没有父节点。

​ 左孩子节点：结点的左指针指向的节点（左侧子节点）是左孩子节点。

​ 右孩子节点：结点的右指针指向的节点（右侧子节点）是右孩子节点。

​ 父节点（双亲节点）：节点的上层节点就是父节点。

​ 兄弟节点：拥有同一个父节点的节点之间就是兄弟结点。

​ 二叉树的高度：从树根到二叉树最远的节点之间经过的边的数量。

​ 二叉树的深度：从根节点到某个节点的唯一路径上经过的边的数量。

​ 子树：一个节点及其所有子孙节点组成的树。

# 前序遍历

​ 在前序遍历中，首先访问根节点，然后递归地对左子树进行前序遍历，最后递归地对右子树进行前序遍历。因此，前序遍历的遍历顺序是：根节点 -> 左子树 -> 右子树。
