# 什么是栈？

​ 栈是一种线性的数据结构，它是一种容器，可以存放数据的容器，栈类似于一个坑，我们只能从栈的最顶端 (坑的口) 往下写入数据，我们往栈中每放一个数据时，新的数据就会在上一次放的数据的上面。

​ 而且我们取数据时也只能从栈顶（坑口）取数据，也就是说，我们最开始存放的数据要到最后才能拿出来，而最后一次存放的数据可以第一次拿出来，这就是栈的 先进后出 原则，这一点与队列的先进先出不一样。

​ 我们可以将栈想象成手枪弹夹。我们往弹夹里压子弹时，新压入的子弹会在上一次压入的子弹的上面，然后开枪是也是弹夹最上面的子弹被打出去，而弹夹最下面的子弹（也就是第一个被压进去的子弹）会最后一个被打出来。这就是栈的例子。

# 栈类的实现

​ 首先我们的栈也是由一个个结点串联起来的，使用我们要先构造栈的结点。

​ 其次是类的实现，栈的基本操作有入栈和出栈，遍历栈，和销毁栈。

# 栈的构造函数实现

​ 虽说栈顶是出入数据的地方，但是我写的栈，它的第一个结点不存放数据，只是起到一个定位作用，类中有一个 Top 指针指向这个结点，每次对栈进行操作时可以通过它直接访问到栈。而真正出入栈的地方在第二个结点处，也就是 Top 结点的尾指针指向的结点。

​ 我们先 new 一个新结点，然后当栈的第一个结点，他不存放数据，因为当前栈是空栈，所以栈顶指针 Top 和栈底指针 End 都指向这个 new 出的结点，并且要将这个结点的尾指针置为空。

# 栈中没有数据的情况下

​ 直接 new 一个结点然后让临时变量 temp 指向这个结点，然后存入数据，并且新结点的尾指针要置为空。然后让栈顶指针 Top 的尾指针指向新结点，最后由于新结点是栈底，所以让栈底指针 End 指向这个新结点。

# 栈中有数据的情况下

​ 让临时变量 temp 指向栈中第一个存放数据的结点，也就是 Top->next 指向的结点。 然后再 new 一个新结点让 Top->next 指向这个新结点，然后存入数据，最后让新结点的尾指针指向 temp 结点即可。

​ 只需释放掉栈顶指针指向的那个用于定位的结点即可。然后将 Top，End，temp 指针都置为空，以免影响其他程序。
